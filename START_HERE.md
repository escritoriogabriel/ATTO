# 🚀 COMECE AQUI - ATTO Transcriber

Bem-vindo ao ATTO! Aqui está como rodar a aplicação em **30 segundos**.

---

## ⚡ 3 Passos Simples

### Passo 1: Instalar Dependências

```bash
pip install -r requirements-streamlit.txt
```

### Passo 2: Rodar a Aplicação

```bash
python run_streamlit.py
```

### Passo 3: Acessar

Abra seu navegador em:
```
http://localhost:8501
```

**Pronto!** 🎉

---

## 📖 O Que Você Pode Fazer

- 🎙️ **Transcrever áudios do WhatsApp** - Exporte a conversa e faça upload
- 📝 **Gerar relatórios** - Integra transcrições no chat original
- 💾 **Baixar resultados** - Arquivo .txt com tudo transcrito

---

## 🆘 Problemas?

### Erro: "ModuleNotFoundError"
```bash
pip install -r requirements-streamlit.txt
```

### Erro: "Connection refused"
Verifique se `run_streamlit.py` está rodando

### Porta já em uso
Mude em `streamlit_app/backend_api.py` (linha 8) e `streamlit_app/main.py` (linha 35)
